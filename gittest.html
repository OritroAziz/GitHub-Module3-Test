(git add .) this adds all files as well as any changes made in that files in the repository

(git add README.txt) this adds the file named README.txt in the repository

(git config -l) to check info and user data

(git commit -m"Comit-1") to commit what you added in file and name it as Commit-1

(git remote add origin https://github.com/https://github.com/OritroAziz/GitHub-Module3-Test) to add the commit to the github repository

(git remote set-url origin https://github.com/OritroAziz/GitHub-Module3-Test) changes the origin url

(git branch -M main) adds main as the git branch

(git pull origin main) used to pull files from local to repository

(git pull origin main --allow-unrelated-histories) use this if you have any separate files you have in repository that you dont have in local

(git push -u origin main) used to push from local to repository

(git remote --v)to check remote urls
